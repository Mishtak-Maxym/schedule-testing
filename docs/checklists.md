## Модуль: Перегляд розкладу групи

### Функціональність
- [ ] Користувач може обрати групу зі списку.
- [ ] Після вибору групи відображається розклад саме цієї групи.
- [ ] Розклад відображається для непарного тижня.
- [ ] Розклад відображається для парного тижня.
- [ ] Користувач може перемикатися між парним і непарним тижнем.
- [ ] Для кожної пари показано назву предмета.
- [ ] Для кожної пари показано час проведення.
