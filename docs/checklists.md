# Чеклісти для тестування

## Модуль: Перегляд розкладу групи

### Функціональність
- [ ] Вибір групи працює.
- [ ] Показується розклад обраної групи.
- [ ] Показується непарний тиждень.
- [ ] Показується парний тиждень.
- [ ] Перемикач тижнів працює.
- [ ] Назва предмета відображається.
- [ ] Час заняття відображається.
- [ ] Викладач відображається.
- [ ] Аудиторія відображається.

### Негативні сценарії
- [ ] Без вибору групи є повідомлення про
