# Чеклісти для тестування

## Модуль: Перегляд розкладу групи

### Функціональність
- [ ] Користувач може обрати групу зі списку.
- [ ] Після вибору групи показується розклад саме цієї групи.
- [ ] Розклад показується для непарного тижня.
- [ ] Розклад показується для парного тижня.
- [ ] Користувач може перемикатися між парним і непарним тижнем.
- [ ] Для кожної пари показано назву предмета.
